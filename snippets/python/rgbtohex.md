# RGB to Hex value conversion

## Code

```python
def rgb_to_hex(r, g, b):
    return ("{:02X}" * 3).format(r, g, b)

print("The hex value for RGB values (100, 255, 167) = #" + rgb_to_hex(100, 255, 167))
```

## Output

```
The hex value for RGB values (100, 255, 167) = #64FFA7
```
